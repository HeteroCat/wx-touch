# 四个功能页实现要求
总体要求：风格要求页面左侧为输入，右侧为结果展示
## 1. 搜索微信公众号：用户在首页输入框输入公众号名称，点击搜索按钮，调用搜索接口，返回公众号列表。
左边白色右边浅橙色，卡片风格，每个卡片展示公众号的信息。
## 2.获取最新文章列表：输入指定公众号的名称，调用获取最新文章接口，返回该公众号的最新文章列表。
左边白色右边浅橙色，瀑布流风格，每个卡片展示文章的信息。
## 3. 提取文章 Markdown 内容：输入指定文章的链接，调用提取文章 Markdown 内容接口，返回该文章的 Markdown 内容。
左边白色右边浅橙色，展示区域为Markdown格式的文章内容。
## 4.关键词搜索文章：在指定公众号的文章中根据关键词进行搜索。输入公众号名称并指定关键词，调用关键词搜索文章接口，返回包含关键词的文章列表。
左边白色右边浅橙色，瀑布流风格，每个卡片展示文章的信息。
